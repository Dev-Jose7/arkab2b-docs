## Descripcion
El sistema consolida ventas semanales para seguimiento comercial.

## Racional
Habilita decisiones de negocio y control de crecimiento.


## Alcance
- Consolidacion semanal de ventas para seguimiento comercial.
- Exportacion de resultados en CSV/PDF.

## No alcance
- Analitica predictiva o modelos de forecasting.
- Dashboard en tiempo real con actualizacion continua.

## Actor(es)
- Administrador comercial u operador de reportes.

## Disparador
- Corte semanal de negocio o solicitud manual autorizada.

## Precondiciones
- Datos de pedidos/ventas disponibles para el periodo.
- Parametros de corte semanal definidos.

## Flujo principal
1. El sistema agrega ventas del periodo semanal.
2. Calcula top de productos y clientes frecuentes.
3. Genera reporte en formato exportable.
4. Publica reporte para consumo operacional.

## Postcondiciones
- Reporte semanal disponible y trazable por fecha/corte.
- Soporte funcional para decisiones comerciales.

## Supuestos y restricciones
- El valor del reporte depende de la completitud de datos de ventas.
- Parametros por pais pueden cambiar el corte operativo.

## Preguntas abiertas
- No hay preguntas abiertas bloqueantes para `MVP`; el calendario exacto por pais/organizacion queda `Parametrizado` por configuracion regional vigente.

## Criterios de aceptacion
- Incluye total ventas, productos mas vendidos y clientes frecuentes.
- Periodicidad semanal.
- Exportable CSV/PDF.

## Dependencias
- FR-004
- NFR-002

## Riesgos

## Notas de implementacion
Definir cortes de semana operativa por pais.
