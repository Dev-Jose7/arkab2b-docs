## Detalle del requisito (tabla)
| Tema | Detalle |
|---|---|
| Descripcion | El sistema genera reporte semanal de SKU con bajo stock segun umbral configurable. |
| Racional | Permite planificar compras y evitar faltantes. |
| Alcance | - Generacion semanal de reporte de productos por abastecer.<br>- Inclusion de umbral configurable y cobertura estimada por SKU. |
| No alcance | - Emision automatica de ordenes de compra a proveedores.<br>- Optimizacion predictiva de demanda. |
| Actor(es) | - Analista de inventario. |
| Disparador | - Corte semanal programado o ejecucion manual autorizada. |
| Precondiciones | - Umbrales de stock definidos por categoria.<br>- Datos de inventario disponibles para el periodo. |
| Flujo principal | 1. El sistema identifica SKU con stock bajo umbral.<br>2. Calcula cobertura estimada y compone el reporte.<br>3. Genera salida en formato CSV/PDF.<br>4. Publica el reporte para consulta operativa. |
| Postcondiciones | - Reporte semanal disponible con fecha de generacion.<br>- Base funcional para planificacion de abastecimiento. |
| Supuestos y restricciones | - Calidad del reporte depende de la calidad de datos de inventario.<br>- La responsabilidad funcional del umbral recae en el area de inventario. |
| Preguntas abiertas | - No hay preguntas abiertas bloqueantes para `MVP`; la calibracion periodica de umbrales queda como `No bloqueante` de operacion continua. |
| Criterios de aceptacion | - Reporte semanal automatico.<br>- Incluye SKU, stock actual, umbral y cobertura estimada en dias.<br>- Umbral configurable por categoria con responsable funcional definido en el area de inventario y registro de auditoria.<br>- Exportable CSV/PDF. |
| Dependencias | - FR-002<br>- NFR-002 |
| Riesgos | - Umbrales mal calibrados. |
| Notas de implementacion | Responsabilidad funcional del umbral explicitada para cierre de ambiguedad. |
